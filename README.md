# CS216-Project
Contributors: Nathan Levine, Athena Wemmert, Daryn Pham, Ian Couture, CJ Frederickson

This repository contains our group's `CS216: Everything Data` final project. In this project, we analyzed NBA salary data and performed an exploritory analysis as to how salary has affected team success over the past decade.

We scraped data from several different databases (BasketballReference, HoopsHype, etc.), used modules like `numpy`, `pandas` and `seaborn` to create, visualize and explain patterns we observed.

Included in this repository are two Jupyter Notebooks: `data.ipynb` containing our data scraping methods, and `code and visualizations.ipynb`, where we applied our data science modules on this scraped data to clean it up and visualize it in several ways.

Our final report can be found in `Analyzing Salary Strategy in the NBA.pdf`
